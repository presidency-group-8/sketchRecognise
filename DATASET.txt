The dataset used for testing in face_models_evaluator.py
is available at :

https://drive.google.com/drive/folders/1knOCSn68Sxv35xtXiF-MnYqJP402jt3O?usp=sharing

The datasets used for model evaluation are:
1.  CASIA face image dataset    --->   21,140 Faces
2.  CUHK face sketches dataset  --->   1712 Sketches
3.  Non face image dataset      --->   5913 Images
4.  Emotions dataset            --->   8100 Face Images and a CSV file containing its emotions
